## Scope

Defines the runtime behavior of the Relationship organ.

This document describes how Relationship operates during the OBSERVE lifecycle.

# Purpose

Relationship derives connections between validated truths and meanings.

Relationship does not create knowledge.

Relationship does not create understanding.

Relationship identifies how validated cognitive outputs interact.
