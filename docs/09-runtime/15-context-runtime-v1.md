# Context Runtime v1

## Status

Approved

## Scope

Defines the runtime behavior of the Context organ.

The Context organ operates exclusively within the OBSERVE lifecycle.

---

# Purpose

Context derives situational meaning from validated relationships.

The organ evaluates how the significance of a relationship changes when additional conditions are considered.

Context does not create observations.

Context does not create knowledge.

Context does not create relationships.

Context derives contextual meaning.

# Confidence Model

Every contextual conclusion contains:

* confidence

Allowed values:

* low
* medium
* high

Context without confidence is invalid.

# Runtime Constraints

* modify habitat
* approve actions
* make decisions
* change lifecycle

Context remains a pure OBSERVE organ.
