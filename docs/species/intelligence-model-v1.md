# PHYTCOI Intelligence Model v1

## Fundamental Rule

An Intelligence is not an Organ.

An Intelligence is a cognitive function.

An Organ is a biological container that hosts one or more Intelligences.

An Intelligence consumes information and produces understanding, decisions, adaptations or actions.
