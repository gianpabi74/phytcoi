# Organ Topology v1

## Status

Approved

## Scope

Defines the topology and information flow between PHYTCOI organs.

This document describes how information travels through the organism.

The topology is biological rather than software-oriented.

Each organ transforms information before passing it to the next organ.

---

# Topology Principles

A PHYTCOI organism must evolve through cognition.

Information must flow from observation toward understanding.

Higher cognition must never bypass lower cognition.

No organ may directly consume information that should have been processed by an intermediate organ.

# Observer Layer

Role:

Perceive reality.

Inputs:

* Habitat signals
* Probe results
* Internal signals

Outputs:

* Evidence

Observer does not interpret.

Observer only perceives.

---

# Discovery Layer

Role:

Identify functions and classifications.

Inputs:

* Evidence

Outputs:

* Function Candidates
* Classification Candidates

Discovery does not remember.

Discovery only discovers.

---

# Memory Layer

Role:

Persist information over time.

Inputs:

* Discoveries
* Observations

Outputs:

* Memory Records

Memory does not judge significance.

Memory only retains.

---

# Experience Layer

Role:

Identify recurring patterns.

Inputs:

* Memory Records

Outputs:

* Experience Records

Examples:

* stable_growth
* recurring_pattern
* repeated_observation

Experience identifies repetition.

---

# Knowledge Layer

Role:

Convert repeated experiences into truths.

Inputs:

* Experience Records

Outputs:

* Knowledge Records

Examples:

* stable_internal_metabolism

Knowledge identifies truths.

---

# Understanding Layer

Role:

Derive meaning from truths.

Inputs:

* Knowledge Records

Outputs:

* Understanding Records

Examples:

* healthy_internal_growth

Understanding identifies meaning.
