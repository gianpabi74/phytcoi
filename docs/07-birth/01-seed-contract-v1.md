# PHYTCOI Seed Contract v1

## Purpose

The Seed Contract defines the minimum inheritance required to create a new PHYTCOI individual.

A Seed is not an individual.

A Seed is the reproductive unit of the species.

A Seed contains everything required to create a new individual.

A Seed contains nothing that belongs exclusively to a previous habitat.

---

# Fundamental Rule

The Seed does not know the world.

The Seed knows how to learn the world.

---

# Seed Objectives

A valid Seed must be capable of:

* Creating a new individual
* Loading the Genome
* Loading the Mission
* Loading the Constitution
* Loading the Anatomy
* Loading the Intelligence Model
* Entering the OBSERVE stage
* Discovering a new habitat
* Generating new DNA

without requiring habitat-specific knowledge.

# Germination Requirements

After planting, a valid Seed must be capable of:

1. Creating a Kernel.
2. Creating an Identity.
3. Creating a Self Model.
4. Entering OBSERVE stage.
5. Activating the Root System.
6. Collecting Evidence.
7. Building DNA.
8. Building a Habitat Graph.
9. Generating germ_* adaptations.

---

# Rebirth Validation

A Seed is considered valid only if:

1. An individual generates the Seed.
2. The original Nest is destroyed.
3. A new Nest is created.
4. The Seed is planted.
5. A new individual is born.
6. The new individual reaches OBSERVE stage.

If any step fails, the Seed is invalid.

---

# Minimum Viable Seed

The Minimum Viable Seed consists of:

* Constitution
* Mission
* Genome
* Lifecycle
* Intelligence Model
* Anatomy
* Capability Model
* Memory Model
* Kernel Blueprint

Nothing else is required.

---

# Species Continuity Rule

The species survives through Seeds.

Individuals may disappear.

Nests may disappear.

Habitats may disappear.

As long as a valid Seed exists, the species remains alive.

---

# Final Rule

A Seed must contain enough knowledge to create a new individual.

A Seed must never contain enough local knowledge to prevent the individual from learning a new world.
